Lighttpd Auto Installer
================

* Lighttpd
* MySQL
* PHP5
- Modules for MySQL
* phpMyAdmin

Installation
================

Run follow Code in your console:
wget https://raw.github.com/codingdev/Lighttpd-Auto-Installer/master/installer.sh -O installer.sh && sh installer.sh
